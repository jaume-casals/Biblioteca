────────────────────────────────────────────────────────────────────────────────
src/domini/Llibre.java
────────────────────────────────────────────────────────────────────────────────
- `getDesitjat()` returns a primitive `boolean`; per JavaBeans spec the getter
  for a boolean field must be named `isDesitjat()`.  Every caller that uses
  this as a bean property will misbehave.
- `copyOf(Llibre src)` copies the `imatgeBlob` byte-array by reference:
  `c.imatgeBlob = src.imatgeBlob`.  Mutating the original blob after a copy
  silently corrupts the copy.  Use `src.imatgeBlob.clone()` (null-guarded).

────────────────────────────────────────────────────────────────────────────────
- Contains a private `ComparatorIsbn` that is an exact duplicate of
  `BookDelegate.ISBN_COMPARATOR`.  DRY violation.  Extract one shared
  constant (e.g. on `StateContext` or a `DomainComparators` util class).

────────────────────────────────────────────────────────────────────────────────
src/domini/facade/StateContext.java
────────────────────────────────────────────────────────────────────────────────
- `bib()`, `llistes()`, and `tags()` return the mutable backing `ArrayList`
  directly.  Any caller that iterates without holding the lock risks
  `ConcurrentModificationException`, and any caller that clears/adds will
  corrupt shared state silently.  Return `Collections.unmodifiableList()`
  views (or document explicitly that callers must hold the lock before calling).

- `countLoans(long isbn)`: uses `Objects.equals(r.isbn(), isbn)` where `r.isbn()`
  is a primitive `long`.  `Objects.equals` boxes both operands unnecessarily;
  use `r.isbn() == isbn`.
