# Biblioteca — Code Compaction Pass (autonomous)

Scope: entire `src/` tree (189 `.java` files), walked file by file. This pass reports
ONLY code-reduction opportunities matching the seven patterns below — no style,
naming, logging, error-message, or correctness findings (those are handled by the
separate review in `tot.txt`). Files not listed had nothing in scope.

Patterns: (1) Repeated logic -> shared helper  (2) Copy-pasted blocks -> loop
(3) Class abstraction  (4) Builder/factory collapse  (5) Inline constant arrays
(6) Switch/if-else -> data table  (7) Anonymous class -> lambda/method ref.

Rough aggregate estimate: ~1,100 lines removable across ~78 files. Line ranges are
indicative; estimates are deliberately conservative.
